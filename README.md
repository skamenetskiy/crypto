# ecrypto

[![Go](https://github.com/skamenetskiy/ecrypto/actions/workflows/go.yml/badge.svg)](https://github.com/skamenetskiy/ecrypto/actions/workflows/go.yml)
[![Go Reference](https://pkg.go.dev/badge/github.com/skamenetskiy/ecrypto)](https://pkg.go.dev/github.com/skamenetskiy/ecrypto) 
[![Go Report](https://goreportcard.com/badge/github.com/skamenetskiy/ecrypto)](https://goreportcard.com/report/github.com/skamenetskiy/ecrypto)

Elliptic Curve crypto library, primarily based
on [go-etherum/crypto](https://github.com/ethereum/go-ethereum/tree/master/crypto) package. The library provides
simplified API for working with EC cryptography. 

## Core features

- Generate keys
- Encrypt
- Decrypt
- Sign
- Verify
